HaythServ-2

Fast - Powerful - Lightweight

HaythServ-2 is designed to be fast and powerful, but also to be able to run on very weak machines. I remade the original HaythServ, without basing this new mod on anything else, it's written in pure C++, and that makes it be more powerful, because what you can't do in Lua (Lua was used in HaythServ very much), you can do that in C++. There's still a lot to do and this mod is under deep development.

Still to-do

There's still a lot to do in HaythServ-2, here's a small list of main-features which I'm gonna add in the next stable version:

  • C++ External modules which may be un-/re-/loaded without any restart.
  • Some more player commands, like #cw, #mapsucks or #votekick.
  • #whois player command to check all names used by an IP-Address/IP-Range

Other features that I'm gonna add somewhen:

  • IRC bot.

Current features

  • Root privilege level, to share common admin privileges with friends, but keeping highest privileges for yourself.
  • C++ Player commands.
  • Accounting system.
  • Flagruns system.
  • Rename command.
  • Stats command.
